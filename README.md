# hearts

线上红心大战。

## start

开启后端：`uvicorn server:app --host 0.0.0.0 --port 8000`      
开启前端：`cd templates && python -m http.server 8080`      
请注意前后端的端口不同，`API_BASE`中使用的是后端端口
